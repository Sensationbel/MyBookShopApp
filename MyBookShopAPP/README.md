Учебный проект, имитирующий Книжный интернет магазин. Используются:

- Spring Boot
- Spring MVC
- Spring Data
- Thymeleaf
- Liquibase
- PostgreSql
- Swagger
- Lombok
- Log4j
- Для сохранения Jwt tokens используется redis:
   Start Redis with docker executing:

        docker run --name my-redis-container -p 7001:6379 -d redis
